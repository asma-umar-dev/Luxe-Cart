# LuxeCart — Beauty & Fashion Store

**Live Site:** [asma-umar-dev.github.io/Luxe-Cart](https://asma-umar-dev.github.io/Luxe-Cart/)

LuxeCart is a premium front-end e-commerce storefront for makeup, skincare, handbags, shoes, and accessories. Built as a single-page site with a curated, editorial feel — considered drops, no clutter.

## ✨ Features

- **Hero & Brand Story** — Seasonal editorial hero section ("The Spring Edit") with trust stats (120+ premium brands, 40k+ happy customers, 4.9★ average rating)
- **Shop by Category** — Browse products across Makeup, Skincare, Shoes, and Handbags
- **New Arrivals & Best Sellers** — Curated product rails
- **Full Product Catalog** — Filterable shop page with:
  - Category filter
  - Price range filter (Rs. 0 – Rs. 25,000)
  - Minimum rating filter
  - Sorting (Featured, Newest, Price Low–High, Price High–Low, Highest Rated)
- **Cart & Wishlist** — Slide-out cart with live subtotal, plus a dedicated wishlist modal
- **Checkout Flow** — Multi-step checkout (Details → Shipping → Review) with order summary and confirmation modal
- **Customer Reviews** — Testimonial section with star ratings and verified buyer badges
- **Newsletter Signup** — Email capture for early access to drops and private sales
- **Contact Section** — Contact form plus email, phone, and studio address (Gulberg, Lahore)
- **Sign In / Account** — Sign in / create account modal for order tracking and wishlist sync
- **Promo Banner** — Free shipping over Rs. 6,000 + `LUXE10` discount code banner

## 🛠️ Tech Stack

- HTML5, CSS3, JavaScript (single-page static site)
- Hosted via **GitHub Pages**
- Responsive, mobile-first layout

## 📁 Getting Started

Clone the repo and open `index.html` directly, or serve it locally:

```bash
git clone https://github.com/asma-umar-dev/Luxe-Cart.git
cd Luxe-Cart
# then open index.html in your browser
# or serve locally:
npx serve .
```
